Installing and Using
MUDGrinder

What Is the MUDGrinder:

The MUDGrinder is a visual editing tool.  It is a pleasant alternative to the command-line interface described in the Archon's Guide (ArchonGuide.html) for creating and maintaining the areas, maps, rooms, monsters, and items inside your MUD.  Instead of typing in commands, you can point and click to design your maps.  Instead of running around with your Archon to get an idea of how things look, you can just point your eyes at the map on the screen.

Installation:

In previous versions of the CoffeeMud Server, the MUDGrinder referred to a stand-alone Java Swing application.  This application, when run, would connect directly to your CoffeeMud server and thereby read and write your areas.  In CoffeeMud 3.0, the MUDGrinder is a built in part of the Administrative Web Site of the CoffeeMud HTTP Web Server.  For more information on this aspect of CoffeeMud, please refer to the Web Server Guide (WebServer.html).  It should be noted that previous versions of the MUDGrinder will not work with CoffeeMud 3.0.  You might even damage something trying.  Don't. 

So, if you have successfully installed and started up your CoffeeMud Server, and the Administrative Web Server has been successfully started as well, then MUDGrinder is already installed!  Good job!

To access MUDGrinder, simply connect to your Administrative Web Site (http://localhost:27777/index.html) and log into the MUDGrinder pages using your Archon administrative account information.  If you have no idea what an Archon Administrative account is, you'll need to start back over again and read the Installation Guide (InstallationGuide.html).

Not Your Mothers MUDGrinder:

The MUDGrinder is easy to use, but it is not forgiving, and it is not for wussies.  If you make a mistake, you will probably pay for it.  If you click the wrong button, you will regret it.  Be careful kiddies!

Using MUDGrinder:

If you've read the Archon's Guide (ArchonGuide.html), and you understand it, you will have no troubles at all.  The MUDGrinder is merely a visual means of accomplishing the same tasks, with the same objects and parameters as are described in the Archon's Guide.

If you have any further questions, or want to know how to undo a mistake,  refer to the previous section.
